A WordPress plugin that let us to insert and image anywhere on you web page.

== Description ==

With WordPress Customizer you can insert an image on your web page.

   * Upload an image.
   * Set the width and height of image.
   * Set the space from the top of the page.
   * set the space from the left of the page.
   * Add a URL to link from image.
   * Set the position of the image (fixed or absolute).

More information and request: https://www.valhallawp.com/plugin/wp-image-anywhere

== Installation ==

1. Visit your admin section and click Plugins -> Add New.  Then search for "WP Image Anywhereâ€.

2. After activate it, from Appearance -> Customize -> Image Anywhere you can insert and set the image settings.
